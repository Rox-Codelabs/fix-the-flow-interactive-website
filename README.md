Deze sprint heb ik me vooral bezig gehouden met het leren van javascript en heb daarmee allerlij interacties op de website gemaakt.
zo is het css only hamburger menu omgezet naar eentje die opent en sluit met javascript en daarom nu ook een transition erop heeft.
de tweede interaction die ik heb gemaakt is de header. zoals ook op de huidige website veranderd de header als je gaat scrollen en veranderd hij weer terug als je helemaal omhoog bent gescrolled.

er is ook een popup als je op zoek bent naar een speciefiek artiekel kun je in deze popup 3 vragen beantwoorden en dan filter hij de artikelen op jou wensen.
de popup is met javascript gemaakt en je kan er doorheen klikken maar de daatwerkelijke filterfunctie werkt nog niet. wat wel werkt is de filtercount en article count die aangeven hoeveel arikelen er in het overzicht zichtbaar zijn en hoeveel filters er zijn toegepast. in de toekomst zouden deze filters die nu in alleen in de popup te zien zijn ook in de linker kolom te zien zijn.
ook was de vraag vanuit de opdrachtgever om de teksten in de artikle cards allemaal automatisch te laten afbreken na een aantal woorden en dan met "..." te laten eindigen in plaats van dit handmatig te doen heb ik dit met javascript gedaan. de filter interactie is uitgewerkt door een user story te schrijven en een wireflow te maken en daarna aan de hand van het javascript 3 stappenplan uitgewerkt.

tijdens de lessen over navigeren en labels heb ik geleerd dat icoontjes labels nodig hebben om duidelijk te maken waar ze voor dienen dit heb ik al hier en daar proberen toe te passen bijvoorbeeld in de popup maar nog niet in de header omdat deze hetzelfde moet blijven als de huidige website.
