## 1. Hamburger Menu
- Het CSS-only menu is omgezet naar een **JavaScript-menu**.
- Vloeiende **open/close transitions** toegevoegd.

## 2. Scrollende Header
- Header verandert van stijl bij scrollen.
- Keert terug naar de oorspronkelijke stijl bij volledig omhoog scrollen.
